### Installation

1. **Download**: Download this repository as a ZIP file.
2. **Extract**: Unzip the folder to a location on your computer.
3. **Load**: 
    * Open Google Chrome and navigate to `chrome://extensions/`.
    * Toggle <u>**Developer mode**</u> (top right corner).
    * Click <u>**Load unpacked**</u> and select the extracted folder.
4. **Open**:
